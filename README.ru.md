## Browser Fingerprint Lab

Очищенная публичная версия прототипа бакалаврской работы по сбору атрибутов
браузерного отпечатка и исследованию их категориальной энтропии.

В репозитории **нет готового набора данных**. При запуске создаётся локальная
SQLite-база в каталоге `data/`, который исключён из Git.

## Что собирается

Только после явного нажатия кнопки локальное приложение получает:

- User-Agent, платформу и язык;
- разрешение экрана и глубину цвета;
- детерминированный canvas-хеш;
- WebGL vendor/renderer, если браузер их раскрывает;
- небольшую выборку доступных шрифтов и список плагинов;
- выбранные HTTP-заголовки;
- признаки cookies, storage, Do Not Track и блокировщика рекламы.

## Локальный запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
python app.py#
```

Открыть `http://127.0.0.1:8080`.

## Docker

```bash
docker compose up --build
```

Локальные образцы сохраняются в `./data`, но не попадают в Git.

## Ограничения и этика

Это учебный исследовательский прототип, а не промышленная система трекинга или
антифрода. Использовать только на своих системах либо с информированным согласием.
Энтропия относится исключительно к локальной выборке. Показатель information gain
оставлен как историческая попарная эвристика и не является строгой оценкой вклада
признака в уникальность пользователя.

Подробности: [docs/PRIVACY.md](docs/PRIVACY.md).
