# Changelog

## 1.0.0 — public cleanup

- removed the historical SQLite dataset and runtime files;
- added automatic creation of a local ignored database;
- replaced request-log parsing and generated static JSON with a JSON API;
- removed jQuery, CryptoJS and third-party ad-block scripts;
- added explicit consent-style collection button;
- replaced the large duplicated result-table code with a data-driven renderer;
- removed pandas and SciPy dependencies in favour of standard-library analysis;
- disabled Flask debug mode by default;
- added Docker Compose, privacy notes and bilingual documentation.
